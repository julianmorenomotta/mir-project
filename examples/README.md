# Audio Examples

This folder contains example audio files generated by our bioacoustic mixture synthesis pipeline.
Each subfolder corresponds to one synthesized session containing two-speaker mixtures from the macaque corpus.

## Folder structure
```
examples/
  mixture_00001/
    mixture.wav      — two-speaker mixture (individuals IO + MU summed)
    IO.wav           — ground truth: individual IO only
    MU.wav           — ground truth: individual MU only
    metadata.json    — session metadata: placement timings, sample rates, split info
```

## Dataset notes

- Source corpus: Japanese macaque vocalizations (*Macaca fuscata*)
- Original recordings sampled at 24,414 Hz, resampled to 44,100 Hz
- Individual calls are shorter than 1 second; sessions of ~10 s are built by
  concatenating calls from two individuals with randomized inter-call gaps
  drawn uniformly from [0.2, 1.0] s and random SNR offsets from [0, 5] dB
- Splits are individual-level: no individual appears in more than one partition

## How these examples relate to the paper

These files illustrate the data pipeline described in Section 3.6.
The system takes mixture.wav and a held-out enrollment clip from one individual
as input, and should produce an estimate close to IO.wav or MU.wav depending
on which individual is queried.
